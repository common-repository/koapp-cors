== Description ==

CORS quick-fix for api calls using the send_headers hook. No settings required.

== Installation ==

1. Upload the uncompressed contents of `koapp-cors.zip` to the `wp-content/plugins/` directory
2. Activate the plugin from the 'Plugins' section in WordPress

== Frequently Asked Questions ==

= What is KingOfApp? =

KingOfApp is the first CMS for mobile Apps. Build your own Apps with our modules and templates, or create and trade with your own ones in our markets. Get into the Kingdom with King of App.

= Why do KingOfApp modules need this plugin? =

Wordpress modules from the KingOfApp market usually need to adapt the headers to prevent CORS issues with the different mobile platforms.
